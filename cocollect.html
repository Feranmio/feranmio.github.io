CoCollect, 2018

Java Web App on Spring MVC Framework

Interacting with debt collection companies over the phone at the time, I realised I dreaded the exchange with the collection agents. After doing research I realised there were a lot of people in a similar predicament that would much rather defer payments or set up payment plans without interacting directly with a collection agent and that led to the birth of CoCollect. As the project progressed I had an increased responsibility to create a debt collection system that protected debtors from unethical collection practices and reduced the emotional distress from being in debt. It was important to also incorporate features that attracted clients and other users to make use of the software.

Dashboard

Central hub to create new cases, add new user data, view existing data relevant to the user including a keyword search feature.

Report Generator

Generate case reports in both PDF and CSV format.

Case Details

Comprehensive information of cases and corresponding debtor.

Income/Expenditure & Payment Plan

Create new Income/Expenditure statements, update Payment Plans with ease and a history of previous updates.

Online Payment

Make card payments online through a secured payment portal.

Analytics

View case analytics to identify trends, make estimates and projections.
